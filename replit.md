# Миллион на Вайб-Кодинге — Презентация

## Описание
Интерактивная веб-презентация для вебинара "Миллион на Вайб-Кодинге" от Кости на Вайбе. 30 слайдов с плавной навигацией и анимациями.

## Структура
- `client/src/pages/presentation.tsx` — Основной компонент презентации, все 30 слайдов
- `client/src/App.tsx` — Роутинг (единственная страница — презентация)
- `client/src/index.css` — Тёмная цветовая схема (purple/cyan градиенты)

## Навигация по слайдам
- Стрелки клавиатуры (влево/вправо)
- Пробел (следующий слайд)
- Кнопки на экране (стрелки по бокам)
- Свайп на мобильных устройствах
- Точки навигации внизу справа (на десктопе)

## Технологии
- React + TypeScript + Vite
- Framer Motion (анимации переходов)
- Tailwind CSS (стилизация)
- Lucide React (иконки)
- Wouter (роутинг)

## Дизайн
- Тёмная тема с фиолетово-голубыми градиентами
- Шрифт: Inter (sans), JetBrains Mono (mono)
- Прогресс-бар сверху
- Адаптивный дизайн (мобильные + десктоп)

## Контент
30 слайдов вебинара:
- Блок 1 (1-5): Крючок и доверие
- Блок 2 (6-11): Сайты и боты
- Блок 3 (12-16): Автоматизации
- Блок 4 (17-19): Стартап + окно возможностей
- Блок 5 (20-22): Структура миллиона
- Блок 6 (23-26): Соцдоказательства + переход
- Блок 7 (27-30): Оффер и закрытие
